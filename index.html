Usually just planner.json at the top level.
Fine-grained token, Contents read/write, scoped to this one repo.
First-time setup
  1. Create a private repo (e.g. planner). The app will create planner.json in it on first save, so you can start empty.
  2. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token.
  3. Repository access → Only select repositories → pick your planner repo.
  4. Permissions → Repository permissions → Contents → Read and write.
  5. Generate, copy, paste above, then Connect & sync.
